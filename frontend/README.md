# 🏘️ UserManagement Frontend

React 19 + TypeScript 物业管理前端，Docker 部署走 Nginx 反向代理。

## 启动

```bash
npm install
npm run dev          # http://localhost:5173
npm run build        # 生产构建 → dist/
```

## 后端

[UserManagement_Spring](https://github.com/ZJW0420/UserManagement_Spring)

## Docker

前端随 `docker compose up -d` 自动构建部署，不需要单独启动。
